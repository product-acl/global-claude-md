# Global CLAUDE.md

Living reference for Claude Code global instructions at Agile Catalyst Labs.

**Live version:** [https://product-acl.github.io/global-claude-md/](https://product-acl.github.io/global-claude-md/)

The source of truth is [`CLAUDE.md`](./CLAUDE.md). Every push to `main` that touches it auto-deploys to GitHub Pages.

## Usage

Symlink or copy into your Claude Code global config:

```bash
# Option A: Symlink (always up to date)
ln -sf "$(pwd)/CLAUDE.md" ~/.claude/CLAUDE.md

# Option B: Copy
cp CLAUDE.md ~/.claude/CLAUDE.md
```
